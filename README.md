# Konmel Website

Zvanična web prezentacija firme **Konmel d.o.o.** iz Beograda.

---

## O projektu

Ovo je statički web sajt razvijen korišćenjem modernih web tehnologija, bez dodatnih framework-a.

Sajt je optimizovan za:

- desktop računare
- tablete
- mobilne telefone

Podrazumevani jezik je **srpski**, uz mogućnost prelaska na **engleski**.

---

## Tehnologije

- HTML5
- CSS3
- Vanilla JavaScript

---

## Funkcionalnosti

- responzivan dizajn
- dvojezičnost (SR / EN)
- SEO optimizacija
- GitHub Pages kompatibilnost
- brza učitavanja
- bez zavisnosti od eksternih biblioteka
